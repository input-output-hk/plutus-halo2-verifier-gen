# Aiken Halo2 Verifier

Aiken implementation of Halo2 zero-knowledge proof verifier for Cardano smart contracts.

## Structure

- `lib/` - Core verifier library modules (BLS12-381 operations, MSM, KZG, polynomial operations)
- `lib/proof_verifier.ak` - Generated circuit-specific verifier (auto-generated by `plutus_gen` tooling)
- `lib/vk.ak` - Generated verification key constants (auto-generated by `plutus_gen` tooling)
- `validators/` - Validator scripts for on-chain deployment

## Building

```bash
aiken check  # Type-check and run tests
aiken build  # Compile to Plutus UPLC
```

## Testing

Some tests require generated code. Run from the repository root:

```bash
cargo run --example equations_test      # Generates equation verification tests
cargo run --example atms                # Generates ATMS circuit verifier
```

Then run tests:

```bash
aiken check
```
